# PREIFMA API

API REST do aplicativo móvel de estudos **PreIFMA** — plataforma desenvolvida como TCC para preparação de candidatos ao Seletivo Técnico do IFMA.

Fornece ao app mobile os recursos de: autenticação JWT, questões com textos e imagens, gamificação (XP, streak, missões, conquistas) e ingestão automatizada de provas via IA (Gemini + PyMuPDF).
